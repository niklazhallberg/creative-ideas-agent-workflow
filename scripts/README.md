# Scripts

Tunna CLI-entrypoints. **Ingen affärslogik här.** All logik bor i
`src/radon/`. Skript i den här mappen ska bara parsa argument, anropa
funktioner i `radon`-paketet, och skriva resultat dit `config.yaml` säger.

Inget i den här mappen är skapat än — filer tillkommer när motsvarande
funktionalitet i `src/radon/` finns och har granskats.
